# Advanced Java Programming Repository

## Overview

This repository contains Java programs focused on advanced core Java concepts and Object-Oriented Programming (OOP).  
It includes practical implementations covering:

- Arrays
- Strings
- Collections Framework
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Interfaces
- Exception Handling
- Wrapper Classes
- Type Casting
- Relationships in OOP

The repository was created as a personal practice workspace for strengthening Java development fundamentals and improving problem-solving skills.
